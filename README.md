# Connect Nest Frontend

> **Turn On Smart Living** — the Connect Nest interface for your smart home hub.

Connect Nest Frontend is a Home Assistant add-on that re-skins your hub with full
Connect Nest branding — colours, fonts, icons, theme, and a custom About panel.
Designed for installation by Connect Nest customers as part of their Connect Nest
smart home setup.

**Connect Nest** transforms homes into smart, connected spaces that enhance
comfort, security, and energy efficiency. Based in Melbourne, Australia.

---

## Features

- 🎨 **Full Connect Nest branding** — sidebar, login, theme, fonts, icons, PWA
- 🌗 **Light and dark themes** built for the Connect Nest brand
- 📱 **iPhone PWA** — "Connect Nest" on the home screen, fullscreen, push notifications
- 🆘 **Built-in help** — one-tap WhatsApp, email, and phone support from the About panel
- 💚 **System health widget** — at-a-glance hub, internet, and backup status
- 📰 **What's new** — release notes inside the app, updated with every release

---

## Install

1. In Home Assistant, go to **Settings → Add-ons → Add-on Store**
2. Tap the **⋮** menu (top right) → **Repositories**
3. Add this URL:
   ```
   https://github.com/roarbis/connectnest-frontend
   ```
4. Click **Add → Close**. Refresh the Add-on Store.
5. **Connect Nest Frontend** will appear in the store. Click it → **Install**.
6. Once installed, click **Start**. Open the **Web UI** to see your branded hub.

Full step-by-step with screenshots: [docs/INSTALL.md](docs/INSTALL.md)
